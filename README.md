## Common commands 
--

`pwd` (**p**rint **w**orking **d**irectory)
Shows what folder you are in 

###### Example:
```bash
> pwd
/home/pavel/Documents
```

`ls` (**l**i**s**t files)
Shows you the files for the specified directory 

###### Example:
```bash
> ls
01_serialExample.R  02_parallelExample.R  testJob.sh
```
